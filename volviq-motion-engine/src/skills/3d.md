## ThreeCanvas Setup

Always wrap 3D content in ThreeCanvas and include proper lighting.

**Incorrect (missing ThreeCanvas wrapper):**

```tsx
<mesh rotation={[0, frame * 0.02, 0]}>
  <boxGeometry args={[2, 2, 2]} />
  <meshStandardMaterial color="#4a9eff" />
</mesh>
```

**Correct (proper ThreeCanvas setup):**

```tsx
import { ThreeCanvas } from "@remotion/three";

<ThreeCanvas>
  <ambientLight intensity={0.5} />
  <pointLight position={[10, 10, 10]} />
  <mesh rotation={[0, frame * 0.02, 0]}>
    <boxGeometry args={[2, 2, 2]} />
    <meshStandardMaterial color="#4a9eff" />
  </mesh>
</ThreeCanvas>;
```

## Premium Lighting Setup

Every 3D scene needs multi-light setup for cinematic depth.

**Incorrect (single flat light):**

```tsx
<ThreeCanvas>
  <ambientLight intensity={1} />
  <mesh>...</mesh>
</ThreeCanvas>
```

**Correct (cinematic multi-light):**

```tsx
<ThreeCanvas>
  {/* Base ambient fill */}
  <ambientLight intensity={0.3} />
  {/* Key light - primary illumination */}
  <directionalLight position={[5, 8, 5]} intensity={0.9} color="#ffffff" />
  {/* Fill light - softens shadows */}
  <directionalLight position={[-3, 4, -2]} intensity={0.3} color="#4f8fff" />
  {/* Rim/back light - edge definition */}
  <pointLight position={[-5, 3, -5]} intensity={0.5} color="#ff6b35" />
  <mesh>...</mesh>
</ThreeCanvas>
```

## Frame-Based Rotation

Use frame directly for smooth continuous rotation.

```tsx
const frame = useCurrentFrame();
const rotationY = frame * 0.02; // Adjust speed with multiplier

<mesh rotation={[0, rotationY, 0]}>
  <boxGeometry args={[2, 2, 2]} />
  <meshStandardMaterial color="#4a9eff" />
</mesh>;
```

## Floating/Hovering Animation

Use sine wave on Y position for organic floating effect.

```tsx
const frame = useCurrentFrame();
const floatY = Math.sin(frame * 0.1) * 0.3; // Amplitude 0.3, speed 0.1

<mesh position={[0, floatY, 0]}>{/* geometry and material */}</mesh>;
```

## Spring-Based Scale Entrance

Use spring() for bouncy 3D object entrances.

```tsx
const scaleProgress = spring({
  frame,
  fps,
  config: { damping: 12, stiffness: 100 },
});

<mesh scale={[scaleProgress, scaleProgress, scaleProgress]}>
  {/* geometry and material */}
</mesh>;
```

## Camera Positioning

Position camera at reasonable distance for scene visibility.

```tsx
<ThreeCanvas camera={{ position: [0, 0, 5], fov: 75 }}>
  {/* scene content */}
</ThreeCanvas>
```

## Animated Camera Orbit

Smoothly orbit camera around the subject.

```tsx
const orbitAngle = frame * 0.015; // Slow continuous orbit
const orbitRadius = 5;
const cameraX = Math.sin(orbitAngle) * orbitRadius;
const cameraZ = Math.cos(orbitAngle) * orbitRadius;
const cameraY = 2 + Math.sin(frame * 0.008) * 0.5; // Subtle vertical drift

<ThreeCanvas camera={{ position: [cameraX, cameraY, cameraZ], fov: 60 }}>
  {/* Scene content - camera automatically looks at origin */}
</ThreeCanvas>
```

## Material Quality - Metalness & Roughness

Tune material properties for realistic surfaces.

```tsx
// Polished metal (chrome, steel)
<meshStandardMaterial color="#c0c0c0" metalness={0.95} roughness={0.05} />

// Brushed metal (aluminum)
<meshStandardMaterial color="#a8a8a8" metalness={0.8} roughness={0.3} />

// Matte plastic
<meshStandardMaterial color="#4a9eff" metalness={0.0} roughness={0.7} />

// Glass-like
<meshStandardMaterial color="#ffffff" metalness={0.1} roughness={0.0} transparent opacity={0.3} />

// Gold
<meshStandardMaterial color="#d4af37" metalness={0.9} roughness={0.15} />
```

## Rim Lighting for Object Edges

Add a backlight to create visible edge definition (rim light).

```tsx
// Position a colored point light behind and to the side of the object
<pointLight
  position={[-4, 2, -4]}
  intensity={0.6}
  color="#4f8fff"
  distance={15}
/>

// Add a second rim light on the opposite side for balance
<pointLight
  position={[4, 1, -3]}
  intensity={0.3}
  color="#ff6b35"
  distance={12}
/>
```

## 3D with 2D Overlay Depth

Combine ThreeCanvas with 2D HTML overlays for rich compositions.

```tsx
<AbsoluteFill style={{ background: "radial-gradient(ellipse at 50% 40%, #1a1a2e 0%, #0a0a0f 100%)" }}>
  {/* 3D Layer */}
  <ThreeCanvas camera={{ position: [0, 0, 5], fov: 60 }}>
    <ambientLight intensity={0.3} />
    <directionalLight position={[5, 8, 5]} intensity={0.8} />
    <mesh rotation={[0, frame * 0.02, Math.sin(frame * 0.01) * 0.1]}>
      <boxGeometry args={[2, 2, 2]} />
      <meshStandardMaterial color="#4f8fff" metalness={0.7} roughness={0.2} />
    </mesh>
  </ThreeCanvas>

  {/* 2D Overlay (text, UI elements) */}
  <AbsoluteFill style={{ pointerEvents: "none" }}>
    <div style={{
      position: "absolute",
      bottom: 80,
      left: 60,
      fontSize: 48,
      fontWeight: 700,
      color: "#e8e6e3",
      textShadow: "0 2px 20px rgba(0,0,0,0.5)",
    }}>
      Product Name
    </div>
  </AbsoluteFill>

  {/* Vignette overlay */}
  <AbsoluteFill style={{
    background: "radial-gradient(ellipse at 50% 50%, transparent 40%, rgba(0,0,0,0.4) 100%)",
    pointerEvents: "none",
  }} />
</AbsoluteFill>
```
